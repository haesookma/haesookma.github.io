Here, I am going to introduce the popular character in Korea, named Ryan.

Idyll’s buit-in component - Scrolling Overlays

The character named Ryan is a popular beloved character in South Korea, newly announced on January 22, 2016.

He is a strong, independent male lion.

At first glance, he looks like a bear, but he is actually a maneless lion.

Ryan was a successor to the throne of Dung Dung Island, but he decided royalty was not for him.

A lot of people misunderstand him because of his expressionless face, but he is the trustworthy mentor who has delicate emotions.

Data Source

The data that I am using is “International Tourism, Number of Arrival” provided by World Bank.

It contains the number of international tourist arriving in each country. In the original data, some of the tables are empty. In this case, I filled out all the empty tables with “0”.

Data Detail

Also, not only does it contain the number of international tourists by each country, but it aslo has several interesting figures, such as the number of world total tourists, the number of tourists based on income, or the number of tourists based on continents.

At the end, I made a plot to select the number of tourists based on income.
